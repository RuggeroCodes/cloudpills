Cloudpills is a completely new website based on a new technology stack.
For years, I'd used a well-known CMS platform based on the Lamp stack, but I was constantly struggling with security issues, updates, and more.
I was looking for something streamlined, simple, low-risk, and where I could also nerdily manipulate the process.
I decided to create a static site using Hugo and base the entire editing process on Visual Studio code and a local container, and finally deploy using GitHub Actions.
